# Agent Guide

Use this wiki as the durable knowledge layer for repository facts, workflows, and architecture summaries. When a task discovers stable project information, record it here instead of only in volatile memory.

Repository policy, guardrails, and operating instructions defer to [`AGENTS.md`](AGENTS.md). The volatile memory bank in [`.kilocode/rules/memory-bank/`](.kilocode/rules/memory-bank/) is local-only and noncanonical; it should summarize current work and point back to pages indexed in [`docs/maintainer-wiki/index.md`](docs/maintainer-wiki/index.md).

For setup and developer workflow, use [`README.md`](README.md), [`DEVELOPMENT.md`](DEVELOPMENT.md), and [`CONTRIBUTING.md`](CONTRIBUTING.md). For file persistence and runtime storage behavior, use [`docs/file-locations.md`](docs/file-locations.md).
