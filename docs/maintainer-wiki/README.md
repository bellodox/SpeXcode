# Maintainer Wiki

This wiki is the canonical durable documentation layer for Carbon Memory in this repository. It stores maintainership guidance, architecture summaries, workflows, and evidence-backed project facts that should persist beyond a single agent session.

Use [`docs/maintainer-wiki/index.md`](docs/maintainer-wiki/index.md) as the catalog for all wiki pages. Session-local context belongs in [`.kilocode/rules/memory-bank/`](.kilocode/rules/memory-bank/), but that volatile memory must summarize and point back to this wiki rather than replace it.

For repository operating policy, defer to [`AGENTS.md`](AGENTS.md). For project setup and contributor workflow, use [`README.md`](README.md) and [`DEVELOPMENT.md`](DEVELOPMENT.md).
