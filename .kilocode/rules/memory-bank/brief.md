#[Memory Bank: Active] Project Brief

This repository is an extension-only Kilo Code monorepo focused on the native VS Code extension product graph. Retained active surfaces remain [`src/`](src), [`webview-ui/`](webview-ui), and required internal packages under [`packages/`](packages); removed product surfaces remain [`apps/`](apps), [`jetbrains/`](jetbrains), [`packages/evals/`](packages/evals), and [`packages/agent-runtime/`](packages/agent-runtime).

Local developer setup has also been simplified to a native-only workflow. Devcontainer, Docker-specific local setup, Nix, direnv, and asdf-style repository setup artifacts were removed in the completed cleanup recorded in durable docs, while this memory bank stays local-only and noncanonical. Carbon Memory in this repository combines this volatile memory bank, the durable maintainer wiki under [`docs/maintainer-wiki/`](docs/maintainer-wiki/), and indexed repository context from entrypoint documents such as [`AGENTS.md`](AGENTS.md) and [`README.md`](README.md); durable facts belong in [`docs/maintainer-wiki/index.md`](docs/maintainer-wiki/index.md).
